Category:
Controllable Image Animation / Motion Transfer

Strengths:

* Image to Video generation
* Landmark driven animation
* Trajectory driven animation
* Hybrid motion control
* Motion transfer from reference signals

* AI Influencer
* Motion Transfer
* Character Animation
* Image To Video Pipeline
* Virtual Human

* Better trajectory control

* Trajectory Animation Module
